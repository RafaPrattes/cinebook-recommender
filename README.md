# 🎬 CineBook Recommender API

![Java](https://img.shields.io/badge/Java-21-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)
![Spring Boot](https://img.shields.io/badge/Spring_Boot-3.2-6DB33F?style=for-the-badge&logo=spring-boot&logoColor=white)
![Status](https://img.shields.io/badge/Status-Developing-blue?style=for-the-badge)

A professional **REST API** built with **Java 21** and **Spring Boot 3** to provide personalized movie and book recommendations. This project focuses on Clean Architecture and modern Java features.

## 🚀 Technical Highlights
* **RESTful Architecture:** Clear endpoints for front-end integration.
* **Modern Java Features:** Extensive use of **Records** for clean, immutable data models.
* **Service Layer Pattern:** Logic decoupled from the controllers for better maintainability.
* **CORS Configured:** Ready to communicate with React/TypeScript applications.

## 🧠 How to use
The API filters media based on genre. You can test it by running the application and accessing:
`GET http://localhost:8080/api/recommendations?genre=Sci-Fi`

## 🛠️ Tech Stack
* **Language:** Java 21 (LTS)
* **Framework:** Spring Boot 3.x
* **Build Tool:** Maven
* **DevOps:** GitHub Actions (CI/CD)
